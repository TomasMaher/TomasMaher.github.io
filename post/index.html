Project 3: Masters Thesis

I chose to do my thesis on the topic of Cybersecurity. I was interested to see how Neutral countries were responding to the ever rising threat of a Cyberwar. I used Text Analysis Methods to compare the manifestos of political parties in several countries to understand if this was an issue that was being prioritised. At a high level this involved: Generating a cybersecurity dictionary to identify statements regarding this topic in different manifestos Translating this dictionary into 10 other languages and using this to extract relevant statements from manifestos Individually translating and categorising over 2,500 statements in 11 languages Locate supporting data on items such as military spending, political alignment, and election wins.

Project 2: NATO Speeches: Code

This was a project I did for during my masters for my Quantitative Text Analysis class. It was completed entirely using R. The following steps were involved: Scraping over 1,000 speeches from the official NATO website using the rvest package Cleaning the data and preparing it for analysis using stringr and quanteda Generating a topic model using the stm package Validating the model against a dictionary based method Plotting the data based on a 28-speech sliding window Exploring the data and drawing conclusions based on this I have included the code used to analyse the data below.
